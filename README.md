# Multiplayer Editor

A multiplayer Geometry Dash mod for building levels with others, made using [enet](https://github.com/lsalzman/enet).

im not really active anymore on this project but i will be updating this sometimes

----------------------------------------------------------

## Installation

1. Install [Geode](https://geode-sdk.org/)
2. Download the latest release from the [Releases page](#)
3. Place the `.geode` file in your `geode/mods` folder
4. Launch Geometry Dash

This mod uses **P2P**, we recommend to use [Hamachi](https://vpn.net/) or similar to create a virtual LAN
